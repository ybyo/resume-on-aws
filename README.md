# Resume on AWS
AWS의 주요 서비스들을 활용하여 배포한 정적 웹 페이지 기반 이력서

[Forrest Brazeal](https://forrestbrazeal.com/)의 [Cloud Resume Challenge](https://cloudresumechallenge.dev/docs/the-challenge/)의 도전과제들을 수행했습니다.
